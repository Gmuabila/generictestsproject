HTML base Tag

When a base URL is defined in the head section of the html document
Any other href in the body of the document link will be search in the URL specified in the base tag.
For example, the base URL here is https://www.w3school.com.
The href below will be searched only in the w3 URL defined in the
base tag above "https://www.w3school.com. If you try to search any other URL
outside of W3 school, it will not be found.
Therefore, the document you are searching for must be a document within W3.
